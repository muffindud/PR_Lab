## Instructions:
### Virtual environment setup
```bash
python3 -m venv venv
```

### Activate virtual environment

Linux:
```bash
source venv/bin/activate
```

Windows:
```bash
venv\Scripts\activate.bat
```

### Install dependencies
```bash
pip3 install -r requirements.txt
```

### Run the app
```bash
python3 main.py
```
